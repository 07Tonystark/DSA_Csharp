C# Gotchas

No Built-in Deque

Use LinkedList with AddFirst/AddLast/RemoveFirst/RemoveLast for sliding window problems.

Max-Heap in PriorityQueue

Negate priorities: Enqueue(x, -x). Or use Comparer.Create((a,b) => b.CompareTo(a)).

jagged vs multidimensional

int[][] for graphs (ragged rows). int[,] for grids — different indexing syntax.

BinarySearch returns ~idx

Negative result means insertion point: idx = ~Array.BinarySearch(arr, x).

HashSet.Add returns bool

false = already exists. Perfect for visited check in one line.

String concat in loops

Use StringBuilder — repeated += is O(n²) total time.

Quick Templates

BFS

var q = new Queue<int>(); var vis = new HashSet<int>(); q.Enqueue(start); vis.Add(start); while (q.Count > 0) { var n = q.Dequeue(); foreach (var nb in graph[n]) if (vis.Add(nb)) q.Enqueue(nb); }

DFS (recursive)

void Dfs(int node, HashSet<int> vis) { vis.Add(node); foreach (var nb in graph[node]) if (!vis.Contains(nb)) Dfs(nb, vis); }

Binary Search

int lo = 0, hi = n - 1; while (lo <= hi) { int mid = lo + (hi - lo) / 2; if (nums[mid] == target) return mid; if (nums[mid] < target) lo = mid + 1; else hi = mid - 1; } return -1;

Monotonic Stack

var st = new Stack<int>(); for (int i = 0; i < n; i++) { while (st.Count > 0 && nums[st.Peek()] < nums[i]) ans[st.Pop()] = nums[i]; st.Push(i); }
